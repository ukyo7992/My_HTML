# login page

## Page lay out
Divide the screen into two section

Left section:
    Use larger image,illustration,gradient,or short message explaining the app

You can also add the app logo and a small description

Right section:
    Place the login form inside a clean card or container

Login form elements
    -A heading such as "Welcome back"
    -A short sentence like "Sign in continue"
    -Email adress field
    -Password fiel
    -show or hide password icon
    -"Remember me" icon
    -"forget the password" link
    -"Continue with google" button
    -A link for users who don't have an account

Design suggestions
    -Rounded input fields
    -Hiding the image section on small devices
    -centering the login card vertically
    -Adding a dark mode
    -Creating focused input styles
    -Adding a password-strength indicator

Design 
    -bg color gray
    -button gold
    -bg of card gradients pink and parple
